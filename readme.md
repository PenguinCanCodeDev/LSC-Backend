# LSC Backend

## Requirements
- Python 3.x
- Other dependencies (see `requirements.txt`)

## Installation
```bash
git clone https://github.com/username/repo-name.git
cd repo-name
pip install -r requirements.txt
```

## Running the project
```bash
python manage.py runserver
```

## API Documentation

- Production
    - Swagger UI: https://domain.com/
    - ReDoc: https://domain.com/redoc/

- Development:
Once the server is running, you can view it at:

    - Swagger UI: http://127.0.0.1:800/
    - ReDoc: http://127.0.0.1:8000/redoc/
